## 엘지 노트북 모델명 확인 방법
~~~~
15ZD970-GX50K
=> 11-22-333-4-55-6-7
=> 15-ZD-970-G-X5-0-K
~~~~
* [1] 화면크기
  * 13 / 14 / 15
* [2] OS 유무
  * Z = Windows 설치됨
  * ZD = Non OS 모델
* [3] 출시년도
  * 960 = 16년 / 970 = 17년 / 980 = 18년 / 990 = 19년
* [4] 배터리 라인업
  * G = 60Wh 대용량 배터리 라인
  * E = 일반 배터리 라인
* [5] CPU 종류
  * 7 = i7 / 5 = i5 / 3 = i3 / 2 = 펜티엄 / 1 = 셀러론
* [6] RAM 용량
  * 0 = 4GB / 4 = 8GB
* [7] 색상
  * K = 화이트
  * BK = 블랙
