## 삼성 노트북 모델명 확인 방법
~~~~
NT900X5N-X58
=> 11-222-33-4-55-6
=> NT-900-X5-N-X5-8
~~~~
* [1] NT = **N** o **t** ebook
* [2] 등급
  * [낮음] 2 < 3 < 5 < 9 [높음]
  * 900 => 9
* [3] 화면 사이즈
  * X5 = 15인치
  * X3 = 13인치
* [4] 타이틀 네임
  * Always = N / Metal = L
* [5] CPU 정보
  * 3 = i3 / 5 = i5 / 7 = i7
* [6] RAM 용량
  * 4 = 4GB / 8 = 8GB
