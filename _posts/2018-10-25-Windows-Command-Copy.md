### CMD 또는 Powershell 에서 사용되는 명령어 입니다.

# Copy 명령어
~~~
COPY [/D] [/V] [/N] [/Y | /-Y] [/Z] [/A | /B ] 원본 [/A | /B]
     [+ 원본 [/A | /B] [+ ...]] [대상 [/A | /B]]
~~~

# Copy 명령어 옵션
~~~
원본         복사할 파일을 지정
/A           ASCII 텍스트 파일임을 나타냄
/B           이진 파일임을 나타냄
/D           대상 파일이 암호화 없이 만들어지도록 허용함
대상         새 파일에 대한 디렉터리와/또는 파일 이름을 지정함
/V           새 파일이 올바르게 기록되었는지 확인함
/N           8.3 이름이 아닌 파일을 복사할 때, 짧은 파일 이름이 있으면
             그 이름을 사용함
/Y           이미 있는 대상 파일을 덮어쓸지를 확인하지 않는다.
/-Y          이미 있는 대상 파일을 덮어쓸지 확인하기 위해 묻는다.
/Z           다시 시작 모드에 있는 네트워크 파일을 복사함
~~~
